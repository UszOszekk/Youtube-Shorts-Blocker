# YouTube Shorts Blocker

Blocks scrolling on YouTube Shorts, preventing navigation via:

- Mouse wheel scrolling
- Arrow down key
- "Next video" button

This extension doesn't hide Shorts you can still watch them, but you can't scroll trough them.

Helps you stay focused and productive.

## How it works

When you're on YouTube Shorts, the extension intercepts scroll and navigation events to keep you focused.

## Instalation

1. clone this repo
2. go to chrome://extensions
3. enable developer mode
4. click the Load unpacked button and select the folder with the extension
